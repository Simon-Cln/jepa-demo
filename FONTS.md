# Fonts

Ce projet utilise **Pangram Sans Rounded** (Free for Personal Use).

Les fichiers de font ne sont pas inclus dans le repo (licence personnelle).

## Installation

Copier le dossier suivant dans `public/` :

```
public/Pangram Sans Rounded - Free for Personal Use/
  PPPangramSansRounded-Bold.otf
  PPPangramSansRounded-Extrabold.otf
  PPPangramSansRounded-ExtraboldItalic.otf
  PPPangramSansRounded-Light.otf
  PPPangramSansRounded-Medium.otf
  PPPangramSansRounded-Regular.otf
  PPPangramSansRounded-Semibold.otf
  PPPangramSansRounded-Thin.otf
  ...
```

Téléchargeable sur : [pangrampangram.com](https://pangrampangram.com/products/pangram-sans)
